# Termux Theme Customization Script

This script allows you to customize Termux themes and hide the welcome message when opening a new session.

## Features:
- **Theme Selection**: Choose from multiple predefined themes.
- **Customization**: Modify the bash prompt with custom symbols and colors.
- **Welcome Message**: Hides the default Termux welcome message.

## How to Use:
1. Clone this repository or download the script.
2. Run the script using the following command:

```bash
bash pika.sh
```
3. You will be prompted to select a theme, and the welcome message will automatically be hidden.

## Change Theme:
To change the theme, simply re-run the script and select a new theme from the menu.
